# Удаление фона\n\nПростой веб-сервис для удаления фона с изображений. Закидываешь JPG, PNG или WEBP - получаешь PNG с прозрачным фоном. На фронте Streamlit показывает исходник и результат рядом.\n\nСтек: FastAPI + Streamlit, под капотом rembg с моделью `birefnet-general` или прямой BiRefNet через PyTorch. Изображения обрабатывает Pillow.\n\n## Запуск\n\n```bash\nconda env create -f environment.yml\nconda activate background-removal\ncp .env.example .env\nuvicorn backend.main:app --reload --port 8000\n```\n\nВо втором терминале поднимаем фронт:\n\n```bash\nconda activate background-removal\nstreamlit run frontend/app.py\n```\n\nПроверить, что бэкенд живой:\n\n```bash\ncurl http://127.0.0.1:8000/health\n```\n\nЕсли окружение уже создано и нужно только обновить зависимости:\n\n```bash\nconda env update -f environment.yml --prune\n```\n\nВ `environment.yml` прописан Python 3.13. Локально гонял на `.venv` - всё работает \n\n\n\n```text\nbackend/     FastAPI, загрузка модели, обработка изображений\nfrontend/    Streamlit\nml/          расчет метрик и разбор ошибок по маскам\ntests/       тесты метрик, ошибок evaluator и интеграционный прогон моделей\n```\n
