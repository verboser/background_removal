# Заметки по реализации и исследованию

Задача: принять одно изображение и вернуть PNG с прозрачным фоном. Я быстро накидал прототип, чтобы проверить метрики.

## Как работает сервис

```text
Streamlit
  -> FastAPI
    -> проверка типа и размера файла
    -> декодирование через Pillow
    -> модель
    -> PNG с альфа-каналом
```

## Датасет

Для оценки взял Oxford-IIIT Pets. Есть маски trimap, можно посчитать честные ошибки на краях (шерсть/хвосты).
